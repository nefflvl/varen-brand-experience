# VAREN® Brand Experience

Brand narrative and identity system for VAREN — elite performance golf.

**Live site:** [varen-brand-experience.pages.dev](https://varen-brand-experience.pages.dev)

## How to update the live site

1. Edit `index.html` in this folder
2. Open **GitHub Desktop**
3. Write a short summary (e.g. "v2 — updated hero section")
4. Click **Commit to main**
5. Click **Push origin**

Site updates automatically in ~30 seconds.

## How to restore a previous version

In GitHub Desktop: **Repository → Tags → right-click a tag → Checkout**. Or ask me to do it.

## Local backups

The `versions/` folder contains HTML snapshots of every completed version. These stay on your Mac only — never uploaded to GitHub or deployed.

## Stack

- **Typography**: Signifier (serif) · DM Sans (sans-serif) · DM Mono (monospace)
- **Grid**: Swiss 12-column modular grid, golden ratio (φ = 1.618) type scale
- **Spacing**: Fibonacci scale (8, 13, 21, 34, 55, 89, 144, 233px)
- **Logo System**: VAREN® Logo Kit mk1 — Half-Y, Thirds, Vertical Golden Ratio, Insignia
- **Deployment**: Cloudflare Pages — static, no build step, auto-deploys on push to `main`
